DEPLOY TO CLOUD (Runs 24/7 without your laptop)

Option 1: Render.com (FREE)
1. Go to https://render.com
2. Sign up with GitHub
3. Create new Web Service
4. Connect your GitHub repo
5. Deploy - it runs forever for free

Option 2: Railway.app (FREE)
1. Go to https://railway.app
2. Sign up
3. Deploy from GitHub
4. Free tier available

Option 3: Heroku (Paid)
1. Go to https://heroku.com
2. Create app
3. Deploy from GitHub
4. Requires paid plan for 24/7

Build command for Render:
npm install --ignore-scripts && npx puppeteer browsers install chrome

Start command for Render:
node server.js

Carrefour proxy environment variables:
CARREFOUR_PROXY_MODE=fallback
CARREFOUR_PROXY_ENDPOINT=https://your-proxy-endpoint
CARREFOUR_PROXY_API_KEY=your_proxy_api_key
CARREFOUR_PROXY_REGION=TR
CARREFOUR_PROXY_TIMEOUT_MS=30000
CARREFOUR_DEBUG=0

Modes:
- off: use direct Puppeteer only
- required: use proxy only
- fallback: proxy first, then direct Puppeteer fallback

Your app will have a URL like:
https://dessert-cafe-manager.onrender.com

Anyone can access it anytime, even when your laptop is OFF.
